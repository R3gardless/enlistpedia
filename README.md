# 입대백과

```
$ git clone https://github.com/kweonminsung/enlistpedia.git
```

### Frontend (React)

```
$ cd frontend
$ npm install
$ npm start
```

### Backend (FastAPI)

- **가상환경 설정**

```
$ python -m venv venv
$ . venv/Scripts/activate
$ pip install --upgrade pip
$ pip install -r requirements.txt
```

- **서버 열기**

```
$ uvicorn main:app --reload --port 3001
```
